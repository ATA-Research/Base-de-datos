# Base-de-datos
Las unidades de las bases de datos se presentan a continuación: 

## Motores/Turbinas
* tipo (4 tiempos / 2 tiempos)
* potencia [W]
* cilindrada [cc]
* peso [Kg]
* rpm [1/min]
* empuje [N]
* diámetro/longitud [m]
* consumo [ml/min]
* vueltas/min/volt [Kv]
* corriente [A]
* resistencia [Ohm]

## Baterías
* capacidad [mAh]
* peso [g]
